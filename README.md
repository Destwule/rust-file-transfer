# Rust TCP File Transfer

A simple terminal-based file transfer program written in Rust.

Both devices must be on the same network and have the program installed.

---

## What I Practiced

- TCP networking
- Basic protocol design
- File streaming in chunks
- Modular Rust code

---

## Features

- Send files over TCP
- Multi-file support
- Device discovery (subnet scanning)

---

## Future Improvements

- Add device names
- Support folder transfer
- Add UI
